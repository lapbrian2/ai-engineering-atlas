# AI Engineering Atlas

An open, citable synthesis of AI engineering — rendered as a reference work, not a tutorial.

Ten topics. Forty concepts. Twenty-three live interactive primitives. All original writing, cross-verified against the field's canonical texts. No videos. No paywalls. No fluff.

---

## Status

**Design prototype — 2026-04-17.** This repository currently contains a single-file design prototype (`index.html`) demonstrating aesthetic direction, motion language, and information architecture. The production build (Nuxt 4 + Nuxt Content + transformers.js) will replace this as development begins.

## Principles

- **Interactive over inert.** Every concept is something you can change, break, and learn from.
- **Real ML in the browser.** The tokenizer runs real BPE. Attention heatmaps use real transformer output. No toy models.
- **Original, cross-verified writing.** Content is synthesized across the canonical AI engineering texts and cited inline.
- **Open forever.** CC BY-NC 4.0 content. MIT code. No tracking. No login. No paywalls.

## Stack (production target)

- Nuxt 4 + Nuxt Content v3
- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger + Lenis (scroll + motion)
- transformers.js (real ML primitives)
- Pagefind (static full-text search)
- Custom CSS (no Tailwind)
- Vercel (static SSG deployment)
